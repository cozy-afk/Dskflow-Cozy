Device Layout Editor — drag screens to arrange how your cursor crosses between them

Devices

Grid

Each tile is one physical display. Devices with several displays get one tile per display, sharing a colour. Drop two tiles edge-to-edge and the cursor will cross between them there.

Arrangement

Snaps to grid · overlapping edges define adjacency

Server config (deskflow)

Each device is one deskflow screen. Multi-display devices are reduced to their primary display's edges for deskflow, since the stock protocol models one screen per machine.
